Задание:

Реализовать телефонный справочник со следующими возможностями:

1. Вывод постранично записей из справочника на экран

2. Добавление новой записи в справочник

3. Возможность редактирования записей в справочнике

4. Поиск записей по одной или нескольким характеристикам

Описание:

Программа состоит из 3-х модулей

1.data_base:

Модуль для работы с json файлом

Состоит из 3-х функций:
- create_database для создания файла формата json для хранения контактов

- check_data для чтения json файла. Функция возвращает содержимое файла

-write_json функция для записи информации в json файл

2.units.py

Модуль для всопмогательных функций

- page_print для постраничного вывода контактов в консоль 

- all_pages функция для подсчета всех страниц при заданном колличестве элементов на странице
 
3.logic.py

Модуль содержит основную логику программы 

-start_menu функция основного меню для обработки команд пользователя

-show_contact функция для отображения всех контактов 

-add_new_contact функция для добавление нового контакта 

-change_contact функция для удаления контакта или изменение данных о контакте 

-search функция поиска контактов по определенным параметрам

4.main.py
Основной модуль для запуска пограммы
